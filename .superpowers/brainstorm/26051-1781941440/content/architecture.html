数据流

用户输入问题 → Vue → WebSocket → ChatController → ChatService
  → ModelAdapter.select(model) → WebScraperEngine (Playwright)
  → 打开 DeepSeek/千问 Web 版 → 输入问题 → 监听 DOM 变化
  → SSE/WebSocket 逐字推回 Vue → 渲染 Markdown

用户点击停止 → WebSocket 发送 cancel → ChatService.cancel()
  → WebScraperEngine 注入 JS 停止生成 → 保留已生成文本

历史搜索 → Vue → GET /api/sessions/search?q=xxx
  → HistorySearchService → H2 FTS 查询 → 返回匹配会话列表
